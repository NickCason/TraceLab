# Test Organization

- `test/utils/` — 16 unit tests for utility modules
- `test/canvas/` — 8 unit tests for canvas drawing functions (including behavioral spy tests)
- `test/integration/` — 3 integration tests (run with `npm run test:integration`)
- `test/components/` — 5 React component tests
- `test/hooks/` — 4 React hook tests

Fixtures shared by integration tests live in `test/fixtures/`.
